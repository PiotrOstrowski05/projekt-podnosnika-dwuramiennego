# Projekt podnośnika samochodowego dwuramiennego

## Opis projektu
Projekt obejmuje opracowanie pełnej koncepcji inżynierskiej, modelu 3D oraz dokumentacji technicznej ręcznego podnośnika samochodowego. Celem projektu było stworzenie konstrukcji spełniającej założone parametry udźwigu przy zachowaniu bezpieczeństwa i norm technicznych.

## Zakres prac
* **Modelowanie 3D:** Wykonane w programie **Siemens NX** (modelowanie części oraz złożeń).
* **Obliczenia inżynierskie:** Analiza analityczna rozkładu sił i obciążeń działających na ramiona, sworznie oraz śrubę napędową.
* **Dobór elementów:** Wymiarowanie części krytycznych oraz dobór elementów znormalizowanych.
* **Dokumentacja 2D:** Przygotowanie rysunków wykonawczych i złożeniowych w module **Drafting** zgodnie z normami ISO/PN.

## Technologie i narzędzia
* **CAD:** Siemens NX
* **Obliczenia:** Metody analityczne (wytrzymałość materiałów)
* **Standardy:** PN-EN ISO

## Zawartość repozytorium
* `/Models` - Pliki części i złożeń (.prt)
* `/Documentation` - Rysunki techniczne w formacie PDF/TIFF
* `/Calculations` - Opis założeń i obliczeń wytrzymałościowych
